Qi Blocks Premium is a selection of premium blocks for the Gutenberg block editor, developed by Qode Interactive.

== Installation ==

= Minimum Requirements =

* WordPress 5.8 or greater
* PHP version 7.0 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.2 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 128 MB or greater

= Installation =

From your WordPress dashboard

1. Download plugin from qodeinteractive.com
1. Go to your WordPress Dashboard -> Plugins -> Add New -> Upload Plugin
2. Upload Qi Blocks Premium
3. Activate Qi Blocks Premium from your Plugins page.
4. Enjoy :)
